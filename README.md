# Sales-Lead-Optimization-through-Data-Cleaning-and-Analytics
Customer Lead Management &amp; Jewelry Sales Analysis Dashboard built in Excel featuring data cleaning, follow-up tracking, KPI analysis, and interactive business insights visualization.

## 📌 Project Overview

This project focuses on analyzing customer lead data for a jewelry business to improve lead tracking, follow-up management, and customer conversion analysis.

The complete project is built inside a single Excel workbook containing:

* Raw customer lead dataset
* Data cleaning process
* Working sheets for preprocessing
* Business analysis sheets
* Dashboard preparation sheets
* Final interactive dashboard

The project demonstrates a complete analytics workflow starting from raw operational data to actionable business insights and dashboard visualization.
